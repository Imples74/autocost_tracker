# AutoCost Tracker

**AutoCost Tracker** — это веб-приложение для учета, анализа и прогнозирования расходов на содержание автомобиля.

Проект разработан на Django и предназначен для автовладельцев, которые хотят контролировать расходы на эксплуатацию автомобиля, анализировать структуру затрат и получать краткую финансовую аналитику.

Каждый пользователь имеет собственный аккаунт и видит только свои автомобили и расходы. Сервис предоставляет удобный интерфейс для добавления автомобилей, регистрации расходов, просмотра статистики и визуализации данных.

## Основные возможности

* регистрация, вход и выход пользователей;
* хранение данных отдельно для каждого пользователя;
* добавление нескольких автомобилей;
* редактирование и удаление автомобилей;
* добавление расходов для каждого автомобиля;
* редактирование и удаление расходов;
* категории расходов;
* фильтрация расходов по автомобилю и категории;
* расчет общей суммы расходов;
* подсчет количества расходов;
* расчет среднего размера расхода;
* определение самой затратной категории;
* прогноз расходов на следующий месяц;
* круговая диаграмма распределения расходов по категориям;
* аналитическая страница с общей статистикой;
* управление данными через Django Admin.

---

## Аналитические возможности

Сервис автоматически рассчитывает основные показатели пользователя:

* общую сумму расходов;
* количество расходов;
* средний размер одного расхода;
* наиболее затратную категорию;
* распределение расходов по категориям;
* прогноз расходов на следующий месяц на основе последних месяцев.

Для визуализации структуры затрат используется библиотека **Chart.js**.
