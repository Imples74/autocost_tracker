# AutoCost Tracker

**AutoCost Tracker** — это веб-сервис для учета, анализа и прогнозирования расходов на содержание автомобиля.

Проект предназначен для частных автовладельцев, которые хотят понимать, сколько денег уходит на эксплуатацию автомобиля, какие категории расходов являются самыми затратными и какую сумму примерно нужно заложить на следующий месяц.

Сервис позволяет пользователю добавлять один или несколько автомобилей, фиксировать расходы по категориям, просматривать статистику, строить графики и получать прогноз ежемесячных затрат на основе накопленных данных.

## Основные возможности

- регистрация и авторизация пользователей;
- добавление одного или нескольких автомобилей;
- редактирование и удаление автомобилей;
- добавление расходов по каждому автомобилю;
- распределение расходов по категориям;
- просмотр списка расходов;
- фильтрация расходов по автомобилю, категории и периоду;
- расчет общей суммы расходов;
- расчет среднего расхода в месяц;
- определение самой затратной категории;
- построение графиков расходов;
- прогноз расходов на следующий месяц;
- управление данными через Django Admin.

---

## Техническая идея проекта

Проект не является простым списком расходов. Основная особенность сервиса — аналитическая обработка данных.

На основе внесенных пользователем расходов система рассчитывает:

- общую сумму затрат;
- структуру расходов по категориям;
- динамику расходов по месяцам;
- средний месячный расход;
- самую затратную категорию;
- прогноз затрат на следующий месяц.

Прогноз рассчитывается как среднее значение расходов за последние доступные месяцы, максимум за 3 месяца.
